# System Architecture & Topology

This folder is designated for structural hardware schematics. Since the current deployment phase utilizes a pure high-fidelity Python Simulation platform to model edge-computing load signatures, physical circuit diagrams are omitted in this release.

### Planned Hardware Topology:
- **MCU:** ESP32-DevKitC (Wi-Fi Enabled)
- **Sensor:** SCT-013-050 Non-Invasive Current Transformer Clamp[cite: 1]
- **Frontend Logic:** 1.65V DC Mid-Bias Analog Signal Conditioning Circuit[cite: 1]
